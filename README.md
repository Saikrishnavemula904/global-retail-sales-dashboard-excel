# Global Retail Sales Dashboard (Excel)

## Project Overview

This project analyzes retail sales performance using Microsoft Excel.

The dashboard was built to provide insights into revenue, profit, customer behavior, product performance, and geographic sales trends.

## Dataset Information

* Sales Transactions: 62,884
* Customers: 15,266
* Products: 2,517
* Stores: 67

## Tools Used

* Microsoft Excel
* XLOOKUP
* Pivot Tables
* Pivot Charts
* Slicers
* Dashboard Design

## KPIs

* Total Revenue
* Total Profit
* Total Quantity Sold
* Total Orders
* Profit Margin

## Dashboard Features

* Monthly Revenue Trend
* Revenue by Category
* Top 10 Products
* Revenue by Brand
* Revenue by Country
* Revenue by Gender
* Interactive Slicers

## Key Insights

* Computers generated the highest revenue.
* United States was the top-performing country.
* Revenue contribution was nearly equal across genders.
* Overall profit margin reached 58.6%.
